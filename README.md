**Nios II 기반, VGA 출력 아날로그 시계**

HW 설계 Term Project

VGA 모니터에 출력되는 아날로그 시계를 설계함. key 또는 button을 입력하면 interrupt가 발생해 아날로그 시계를 사용자가 직접 설정 가능하도록 함.  

해당 기능으로는 알람기능, 모래시계 기능, 시간 설정, 나침반 기능 등을 수행. 

나침반 기능은 직접 지자기 센서를 연결하여 모니터에 시계와 같이 출력되도록 함.
